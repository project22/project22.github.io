Travel Posts

This is a simple app that works well on both mobile and on the desktop. It allows users to read and post comments based on geolocation, and view images taken near to location.

The backend is built in Ruby on Rails, and uses postgreSQL and ActiveRecord for the database. The mapping is done in Leaflet and the majority of interactivity is built on the front end using jQuery and Bootstrap UI framework.

In Peak Season

This project was inspired by my experience of great seasonal food during my travels overseas. Most cultures in the world source food locally and seasonally.

This app locates the user's geolocation, and then looks up all local food items that are in season nearby. The user can then find recipes that use those ingredients, and find the nearest farmers markets to buy at. The app has responsive design, and works well on mobile devices.
